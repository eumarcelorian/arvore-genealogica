# Árvore Genealógica em Java

Projeto desenvolvido para a disciplina de Estrutura de Dados utilizando a classe DefaultMutableTreeNode da biblioteca javax.swing.tree.

O sistema representa uma árvore genealógica com 5 níveis hierárquicos da Rainha Elizabeth II, utilizando conceitos de árvores, hierarquia e recursividade em Java.

# Objetivo do Projeto

O objetivo desta atividade é praticar:

* Estruturas de dados em árvore;
* Organização hierárquica de informações;
* Recursividade;
* Manipulação de nós em Java;
* Percurso em árvores utilizando recursão.

# Tecnologias Utilizadas

* Java
* javax.swing.tree.DefaultMutableTreeNode
* IDE IntelliJ IDEA
* Programação Orientada a Objetos
* Recursividade

# Estrutura da Árvore

A árvore foi organizada em 5 níveis hierárquicos:

| Nível | Descrição |
|---|---|
| 1 | Pessoa principal (raiz da árvore) |
| 2 | Pais |
| 3 | Avós |
| 4 | Bisavós |
| 5 | Trisavós |

A estrutura segue uma organização hierárquica onde cada nó representa um ancestral da pessoa escolhida.

# Exemplo de Saída

Rainha Elizabeth II
    Pai: Rei George VI

            Bisavô Paterno: Rei Edward VII

                Trisavó Paterna: Rainha Victoria
